# Most Native — стартер

## Что это

Стартовый набор для миграции веб-мессенджера «Мост» на нативное мобильное приложение через Claude Code.

## Содержимое

- **SETUP_NATIVE.md** — пошаговая инструкция для тебя (что установить, как создать GitHub репо, куда положить файлы)
- **CLAUDE.md** — промпт для Claude Code (план миграции по 6 фазам, правила работы)
- **web-source/** — папка для распаковки последнего ZIP веб-версии (`most_v3_8_4.zip`)

## С чего начать

1. **Прочитай SETUP_NATIVE.md от начала до конца**
2. Сделай шаги 1-4 из инструкции (установка Node.js, создание GitHub репо, копирование файлов, скачивание Firebase config)
3. Запусти Claude Code: `cd /Users/rl/most && claude`
4. Claude Code прочитает CLAUDE.md и начнёт Фазу 0

## Порядок копирования файлов

```
/Users/rl/most/                       ← создать папку
├── CLAUDE.md                         ← из этого стартера
├── google-services.json              ← скачать из Firebase Console
└── web-source/                       ← из этого стартера
    └── (распаковать сюда most_v3_8_4.zip)
```

## Сколько времени займёт

- Подготовка (твои шаги): **30-60 минут**
- Фаза 0 (Claude Code): **30 минут**
- Фазы 1-6 (Claude Code): **15-25 сессий по 1-2 часа** = 1-3 недели работы

После Фазы 6 у тебя будет APK файл, который можно установить на любой Android.
